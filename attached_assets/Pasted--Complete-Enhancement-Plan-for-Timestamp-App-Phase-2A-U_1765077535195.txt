### **Phase 2A: User Requirements** (Priority: HIGH)

#### 1. **Zoom Controls + Slider**
- Add zoom in/out buttons (+ / - buttons)
- Add zoom slider (0.1x to 5x range)

#### 2. **Image Position Adjustment Sliders**
- Horizontal position slider (X-axis adjustment)

- Reset button to center image (0, 0)
- Visual indicators showing current position values

#### 3. **Save/Load Preset System**
- Save current settings as named preset (localStorage)
- Load previously saved presets
- Delete unwanted presets
- Preset includes:
  - Time/Date/Location text
  - Font family, size, alignment
  - Text color, shadow color
  - Overlay positions
  - (Optional: zoom level and image position?)
- UI: Dropdown or list to select presets + Save/Delete buttons

#### 4. **Responsive Header Layout**
- Make header buttons flexible/wrap on narrow viewports
- Stack buttons vertically or in multiple rows on mobile
- Ensure all controls remain accessible on small screens
- Breakpoint: ~640px or when buttons start to overflow

#### 5. **Shadow Intensity Control**
- Add shadow blur slider (0-20px range)
- Add shadow offset X/Y sliders
- Visual preview of shadow effect
- Default: current settings (1px offset, 2px blur)

### **Phase 2B: Additional Recommendations** (Priority: MEDIUM)

#### 6. **Helper Buttons for Time/Date**
- "Use Current Time" button - auto-fills current time
- "Use Current Date" button - auto-fills current date
- "Use Current Location" button (if geolocation API available)
- Format options (12h/24h for time, different date formats)

#### 7. **More Font Options**
- Add 10-15 more web-safe fonts
- Or integrate Google Fonts (popular options: Roboto, Open Sans, Lato, etc.)
- Font preview in dropdown

#### 8. **Style Presets (Quick Templates)**
- Pre-configured style presets:
  - **Classic**: White text, black shadow, serif font
  - **Modern**: Bold sans-serif, minimal shadow
  - **Minimal**: Small text, light shadow, clean look
  - **Bold**: Large text, heavy shadow, impact font
  - **Vintage**: Sepia colors, decorative font
- One-click apply preset
- User can still customize after applying preset

#### 9. **Enhanced Shadow Controls**
- Shadow spread control
- Multiple shadow layers option
- Shadow presets (Soft, Hard, Glow, Outline)

#### 10. **Export Options**
- Image quality selector (Low/Medium/High/Maximum)
- File format options (PNG/JPEG/WebP)
- File name customization before download
- Option to include/exclude specific overlays in export

#### 11. **Undo/Redo Functionality**
- History of changes
- Undo/Redo buttons in header
- Keyboard shortcuts (Ctrl+Z / Ctrl+Y)

#### 12. **Overlay Management**
- Show/hide individual overlays (toggle visibility)
- Lock overlay position (prevent accidental moves)
- Duplicate overlay functionality
- Add custom text overlays (beyond Time/Date/Location)
